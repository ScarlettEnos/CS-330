# CS-330
CS 330 Portfolio

For this project, I approached designing my scene by first thinking about the overall goal and then breaking it into smaller pieces that I could build one at a time. 
I looked at the reference image and thought about which simple shapes I could combine to recreate each object. Instead of trying to complete everything at once, I focused 
on modeling one object, adjusting it until it looked right, and then moving on to the next. As I created the scene, I made small changes to each object’s size, position, 
and rotation until everything felt proportional and made sense to me. I also organized my code by creating reusable functions for setting transformations, materials, and textures, which kept me from
repeating large sections of code. This made it easier to make changes later because I only had to update small pieces of my existing code. Learning through trial and error was a big part of my process, 
especially when adjusting lighting and camera movement, since most elements required testing and adjusting multiple times. Over the milestones, my approach became more thoughtful and organized, 
and I learned to make smaller adjustments instead of large changes all at once. This project strengthened my patience and gave me a greater appreciation for how much 
planning and detail goes into creating even a simple 3D scene. It also showed me how much creativity goes into programming. Although I do not plan on using these skills in my future career, 
this project made me appreciate all the digital media I enjoy even more.
